# Cold Dominant Power Retheme — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use the-controller-executing-plans to implement this plan task-by-task.

**Goal:** Replace all Catppuccin Mocha hardcoded colors with CSS custom properties using a monochromatic B&W palette, add Geist font family, and add frosted-glass modal overlays. Pure visual — no logic changes.

**Architecture:** Define all design tokens as CSS custom properties in `app.css :root`. Replace every hardcoded hex/rgba color in ~30 component `<style>` blocks with `var()` references. Install Geist font package. Add `backdrop-filter: blur()` to modal overlays.

**Tech Stack:** CSS custom properties, Svelte scoped styles, `geist` npm package, xterm.js theme config.

## Color Mapping Reference

Every replacement in this plan follows this mapping. When a Catppuccin value appears, use the token based on **CSS property context**:

### Backgrounds

| Old Value | Context | New Token |
|-----------|---------|-----------|
| `#11111b` | body, terminal, deepest bg | `var(--bg-void)` |
| `#181825` | secondary panels, summary pane | `var(--bg-base)` |
| `#1e1e2e` | sidebar, modal bg, panels | `var(--bg-surface)` when panel, `var(--bg-elevated)` when modal |
| `#313244` | input bg, hover bg, badge bg | `var(--bg-hover)` |
| `#45475a` | active/selected bg, stronger hover | `var(--bg-active)` |
| `#585b70` | elevated hover | `var(--bg-active)` |

### Borders

| Old Value | New Token |
|-----------|-----------|
| `#313244` (as border) | `var(--border-default)` |
| `#45475a` (as border) | `var(--border-default)` |
| `rgba(49, 50, 68, *)` | `var(--border-subtle)` or `var(--border-default)` |

### Text

| Old Value | New Token |
|-----------|-----------|
| `#cdd6f4` | `var(--text-primary)` |
| `#bac2de` | `var(--text-primary)` |
| `#a6adc8` | `var(--text-secondary)` |
| `#7f849c` | `var(--text-secondary)` |
| `#6c7086` | `var(--text-secondary)` |
| `#ffffff` (as text) | `var(--text-emphasis)` |
| `#1e1e2e` (as text on light bg) | `var(--bg-void)` |

### Status / Semantic Colors

| Old Value | New Token |
|-----------|-----------|
| `#89b4fa` (blue — focus/accent) | `var(--focus-ring)` when focus, `var(--text-emphasis)` when accent |
| `#a6e3a1` (green) | `var(--status-idle)` |
| `#f9e2af` (yellow) | `var(--status-working)` |
| `#f38ba8` (red/pink) | `var(--status-error)` |
| `#eba0ac` (light pink hover) | `var(--status-error)` with opacity or lighter |
| `#fab387` (orange) | `var(--status-working)` |
| `#89dceb` (cyan) | `var(--text-secondary)` |
| `#cba6f7` (violet) | `var(--text-secondary)` |
| `#f5e0dc` (rosewater/cursor) | `#ffffff` |

### Overlays / RGBA

| Old Value | New Token |
|-----------|-----------|
| `rgba(0,0,0, 0.5-0.7)` overlay | `rgba(0, 0, 0, 0.7)` + `backdrop-filter: blur(16px)` |
| `rgba(137,180,250, *)` blue tints | `rgba(255, 255, 255, *)` at lower opacity |
| `rgba(166,227,161, *)` green tints | `rgba(74, 158, 110, *)` at same opacity |
| `rgba(243,139,168, *)` red tints | `rgba(196, 64, 64, *)` at same opacity |

## Task 1: Migration Completeness Test

**Files:**
- Create: `src/lib/__tests__/retheme-migration.test.ts`

**Step 1: Write the failing test**

This test ensures no old Catppuccin hex values remain in any `.svelte` file's `<style>` block.

```ts
import { describe, it, expect } from "vitest";
import { readFileSync } from "fs";
import { globSync } from "glob";
import { resolve } from "path";

const OLD_CATPPUCCIN_COLORS = [
  "#11111b",
  "#181825",
  "#1e1e2e",
  "#313244",
  "#45475a",
  "#585b70",
  "#6c7086",
  "#7f849c",
  "#a6adc8",
  "#bac2de",
  "#cdd6f4",
  "#89b4fa",
  "#f38ba8",
  "#a6e3a1",
  "#f9e2af",
  "#fab387",
  "#89dceb",
  "#cba6f7",
  "#eba0ac",
  "#f5e0dc",
];

describe("retheme migration", () => {
  it("no Catppuccin hex values remain in component styles", () => {
    const root = resolve(__dirname, "../../..");
    const files = globSync("src/**/*.svelte", { cwd: root });
    const violations: string[] = [];

    for (const file of files) {
      const content = readFileSync(resolve(root, file), "utf-8");
      const styleMatch = content.match(/<style[\s\S]*?>([\s\S]*?)<\/style>/);
      if (!styleMatch) continue;
      const styleBlock = styleMatch[1].toLowerCase();

      for (const color of OLD_CATPPUCCIN_COLORS) {
        if (styleBlock.includes(color)) {
          violations.push(`${file}: still contains ${color}`);
        }
      }
    }

    expect(violations).toEqual([]);
  });

  it("CSS custom properties are defined in app.css", () => {
    const root = resolve(__dirname, "../../..");
    const css = readFileSync(resolve(root, "src/app.css"), "utf-8");
    const requiredTokens = [
      "--bg-void",
      "--bg-base",
      "--bg-surface",
      "--bg-elevated",
      "--bg-hover",
      "--bg-active",
      "--border-subtle",
      "--border-default",
      "--text-emphasis",
      "--text-primary",
      "--text-secondary",
      "--text-tertiary",
      "--status-idle",
      "--status-working",
      "--status-error",
      "--status-exited",
      "--focus-ring",
    ];

    for (const token of requiredTokens) {
      expect(css).toContain(token);
    }
  });
});
```

**Step 2: Run test to verify it fails**

Run: `npx vitest run src/lib/__tests__/retheme-migration.test.ts`
Expected: FAIL — old colors exist everywhere, CSS tokens not defined yet.

**Step 3: Commit**

```css
@import "geist/dist/fonts/geist-sans/style.css";
@import "geist/dist/fonts/geist-mono/style.css";

:root {
  /* Backgrounds — luminance ladder */
  --bg-void: #000000;
  --bg-base: #0a0a0a;
  --bg-surface: #141414;
  --bg-elevated: #1c1c1c;
  --bg-hover: #242424;

  --border-subtle: rgba(255, 255, 255, 0.06);
  --border-default: rgba(255, 255, 255, 0.10);

  /* Text — luminance as hierarchy */
  --text-emphasis: #ffffff;
  --text-primary: #e0e0e0;
  --text-secondary: #777777;
  --text-tertiary: #4a4a4a;

  /* Status — the ONLY chromatic elements */
  --status-idle: #4a9e6e;
  --status-working: #b8a43c;
  --status-error: #c44040;
  --status-exited: #555555;

  /* Focus */
  --focus-ring: #ffffff;

  --font-sans: "Geist", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
  --font-mono: "Geist Mono", "JetBrains Mono", "Fira Code", monospace;
}

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: var(--font-sans);
  background: var(--bg-void);
  overflow: hidden;
  color: var(--text-primary);
}
```
